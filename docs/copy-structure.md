# コピーライティング構造パターン

> ファーストビューのコピー（文章）の型集。
> 業種やターゲットに合わせて選んでください。

---

## 型1：問いかけ→解決 型

**構造**
```
[カテゴリタグ]
悩みを代弁する問いかけ（共感）
  ↓
解決策としてのサービス名/コピー
  ↓
安心させるフォローコピー
```

**例（弁護士事務所）**
```
[消費者金融] [カードローン] [リボ払い]

借金・返済の不安が尽きない...
自己破産しかない！と思い込んでいませんか？

    相談料 0円  何度でも

あなたを責めることはありません。
まずは一番つらいことだけ聞かせてください。
```

**使いどころ**：悩みが深い業種（法律、医療、転職、借金）

## 型2：断言→根拠 型

**構造**
```
[サブコピー（前振り）]
  ↓
強い断言のメインコピー（大文字）
  ↓
根拠・理由（なぜそう言えるか）
  ↓
[信頼バッジ]
```

**例（債務整理事務所）**
```
「債務整理」でどこがいいのか迷ったら
東京総合法務事務所

完済まで
徹底サポート！

借金問題を中心に解決！
だから債務整理に強い。

[ノーリスク 相談料0円] [費用も安心 分割払い] [家族や職場 バレずに]
```

**使いどころ**：自信・権威を見せたい業種（士業、コンサル、BtoB）

## 型3：世界観→ブランドメッセージ 型

**構造**
```
[ブランド名 × コラボ名]
  ↓
ビジュアル（写真がメイン）
  ↓
キャンペーン名（装飾的タイポグラフィ）
  ↓
ブランドの想いを伝えるコピー
```

**例（ファッションブランド）**
```
SAKURA MIYAWAKI × Couturier

愛編む with YOU

編み物『愛』を世界に広げ続ける、
宮脇咲良さんにクチュリエからラブコール！

編み物は、あなたとわたしをつなぐもの。
わたしたちの想いは、いつもあなたと。
```

**使いどころ**：ブランディング重視（ファッション、コスメ、ライフスタイル）

## 型4：数字ファースト 型

**構造**
```
[インパクトのある数字（超大文字）]
  ↓
数字の説明（何の数字か）
  ↓
サービス説明コピー
  ↓
[CTA]
```

**例（プログラミングスクール）**
```
    98.5%

転職成功率

未経験から最短3ヶ月でエンジニアへ。
あなたの人生を変える学習プログラム。

[無料カウンセリングを予約]
[受講料0円で開始] [転職保証あり] [24時間サポート]
```

**使いどころ**：実績・数字に自信がある業種（スクール、SaaS、通販）

## 型5：ビフォーアフター 型

**構造**
```
こんなお悩みありませんか？
  ✗ 悩み1
  ✗ 悩み2
  ✗ 悩み3
  ↓（矢印や区切り線）
解決後の理想の姿（メインコピー）
  ↓
[CTA]
```

**例（エステサロン）**
```
こんなお悩みありませんか？
  ✗ 何をしても痩せない
  ✗ 食事制限が続かない
  ✗ リバウンドの繰り返し

      ↓ ↓ ↓

  -15kg の自分に出会える

最新マシン × 管理栄養士の食事指導で
"最後のダイエット"を始めませんか？

[初回体験 500円で申し込む]
```

**使いどころ**：悩みが具体的でリストにできる業種（美容、健康、修理）

## 型6：季節/イベント 型

**構造**
```
[ナビ/タブ]
  ↓
フルビジュアル
  ↓
キャンペーンタイトル（ロゴ的デザイン）
サブコピー（季節感）
  ↓
[SCROLL]
```

**例（アパレル年末キャンペーン）**
```
[PART 1]  [PART 2]

Thanks to 2025
一年の終わりに感謝を届けよう

      SCROLL ↓
```

**使いどころ**：季節キャンペーン、セール、記念日LP

## コピーの鉄則

### 1. 一文は短く
- NG：「当事務所は借金問題に悩むすべての方に対して完済まで徹底したサポートを提供します」
- OK：「完済まで、徹底サポート。」

### 2. ターゲットの言葉を使う
- NG：「債務整理の最適解をご提案」（専門用語）
- OK：「借金、減らせます。」（ユーザーの言葉）

### 3. 強調ワードを決める
- 1つのコピーの中で「最も見てほしい1〜2語」を決める
- その語だけ色・サイズ・太さを変える
- 例：「完済**まで**徹底**サポート**！」→ 「サポート」を緑色に

### 4. CTA文言は「動詞 + メリット」
- NG：「送信」「申し込み」
- OK：「無料で相談する」「今すぐ500円で体験する」
- ボタン周囲に「30秒で完了」「しつこい営業なし」等のマイクロコピー

### 5. 句読点を効果的に使う
- 体言止めで力強く：「完済まで、徹底サポート。」
- 三点リーダーで余韻：「不安が尽きない...」
- 疑問形で巻き込み：「と思い込んでいませんか？」

## AIが書きがちなNGコピーとその修正

### NG1：ビジネス文体
```
✗ 当事務所は、お客様の借金問題を解決するため、最適なソリューションをご提供いたします。
○ 完済まで、徹底サポート。
```
**問題**：主語が「当社/当事務所」、「ご提供」「ソリューション」等のビジネス用語
**対策**：ユーザー視点で、短く、体言止めで

### NG2：情報を全部詰め込む
```
✗ 債務整理・任意整理・個人再生・自己破産のご相談なら実績豊富な当事務所へ
○ 借金、減らせます。
```
**問題**：SEOキーワードを並べたような一文。FVのコピーは「感情」を動かすもの
**対策**：1つのメッセージに絞る。詳細はサブコピーや下層で

### NG3：曖昧で刺さらない
```
✗ あなたの未来をサポートします
✗ 私たちと一緒に、新しい一歩を
○ 自己破産しかない！と思い込んでいませんか？
```
**問題**：抽象的すぎて何のサービスかわからない
**対策**：具体的な悩み・数字・行動を入れる

### NG4：英語コピー
```
✗ Your Best Solution for Debt Problems
○ 完済まで、徹底サポート。
```
**問題**：日本のLPユーザーは英語コピーを読まない（装飾的に小さく使うのはOK）
**対策**：メインコピーは必ず日本語。英字はサブ要素やロゴの装飾に限定
