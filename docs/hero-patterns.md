## パターン A：権威キービジュアル型

```
┌─────────────────────────┐
│  ■ ヘッダーバー（紺/黒）  │
│  サブコピー               │
│  会社名                   │
├─────────────────────────┤
│                           │
│     ┌──────────┐         │
│     │  人物写真  │         │
│     │  （中央）  │         │
│     └──────────┘         │
│                           │
│   ██ メインコピー ██      │
│   （超大文字・写真に重なる）│
│                           │
│   サブコピー（小さめ）     │
│                           │
├─────────────────────────┤
│ [バッジ1] [バッジ2] [バッジ3] │
└─────────────────────────┘
```

**特徴**
- 人物写真が画面中央に大きく配置
- メインコピーが写真に重なるほど大きい
- 上部にダークカラーのヘッダーバー
- 下部に信頼バッジが横並び

**向いている業種**：士業、コンサルティング、BtoBサービス
**参考**：債務整理LP（画像1）

```
┌─────────────────────────┐
│                           │
│  「こんなお悩みありませんか？」│
│                           │
│  ✗ 悩み1                  │
│  ✗ 悩み2                  │
│  ✗ 悩み3                  │
│                           │
│  ─────── ↓ ───────       │
│                           │
│  ██ 解決策（メインコピー）██│
│                           │
│     ┌──────────┐         │
│     │  人物/商品  │         │
│     └──────────┘         │
│                           │
│  [CTA ボタン]             │
│  [バッジ1] [バッジ2]       │

```

**特徴**
- 上半分で「悩み」を列挙して共感
- 下半分で「解決策」として自社サービスを提示
- 悩み→解決の縦の流れが自然に読ませる
- チェックマークやアイコンで視認性を高める

**向いている業種**：エステ、ダイエット、スクール、転職、修理サービス

## パターン G：エディトリアル型

```
┌─────────────────────────────────┐
│  TOKACHI LIGHT.                  │ ← 巨大英字（画面幅80%以上）
│                                   │
│  ┌─────┐   ┌──────────┐         │
│  │ロゴ  │   │          │  縦書き  │
│  │50TH  │   │  人物写真  │  キャッチ│
│  │会社名 │   │ （中央）  │  コピー  │
│  └─────┘   │          │         │
│  ● ● ● ●   └──────────┘  ┌───┐ │
│  ナビ（縦） ← 幾何学シェイプ重なり →│記念│ │
│                                └───┘ │
└─────────────────────────────────┘
```

**特徴**
- 画面上部に巨大な英字タイポグラフィ（80px〜120px）
- 人物写真に幾何学的なカラーシェイプ（丸、三角、不定形）を重ねる
- 日本語コピーは**縦書き**で配置（writing-mode: vertical-rl）
- ロゴ・記念バッジ・ナビが写真の左右に散りばめられる
- 余白を活かした「雑誌の表紙」的レイアウト

**CSSの肝**
- `writing-mode: vertical-rl` で縦書きテキスト
- 幾何学シェイプは `clip-path` または `border-radius: 50%` + absolute配置
- 巨大英字は `font-size: clamp(48px, 10vw, 140px)` でレスポンシブ
- 人物写真の `z-index` をシェイプより高くして重なりを作る

**レスポンシブ挙動**
- モバイル：英字タイトルは上部に横書き、縦書きコピーはそのまま維持
- タブレット以上：全要素が1画面に収まるように配置

**向いている業種**：採用サイト、コーポレートサイト、周年記念、建設・製造業
**参考**：音更電設 50周年サイト

## パターン H：商品×感情訴求型

```
┌─────────────────────────────┐
│                               │
│  ┌──────────┐    縦書き       │
│  │          │    メイン       │
│  │  人物写真  │    コピー       │
│  │（感情的）  │   （大きめ）    │
│  │ 母+赤ちゃん│               │
│  └──────────┘               │
│                               │
│  ┌──吹き出し──┐              │
│  │ 信頼テキスト │              │
│  └───────────┘              │
│                               │
│  サービス説明テキスト            │
│                               │
│  ┌─────┐                    │
│  │商品画像│ ← 実物パッケージ   │
│  └─────┘                    │
│                               │
│  詳細コピー（成分・特徴等）      │
└─────────────────────────────┘
```

**特徴**
- 人物写真が「感情」を伝える（母子の触れ合い、笑顔、安心感）
- メインコピーは**縦書き**で右側に配置
- 吹き出し・リボン風バッジで信頼要素（「〇〇が企画・開発」等）
- 実物の商品パッケージ写真を配置
- 情報密度が高い（コピー + 写真 + 商品 + バッジが1画面に）

**CSSの肝**
- 縦書きコピー `writing-mode: vertical-rl; font-size: clamp(24px, 6vw, 48px)`
- 吹き出しは `border-radius` + `::after` で三角矢印
- 商品画像は `filter: drop-shadow()` で浮き感
- 背景は淡い暖色系グラデーション

**レスポンシブ挙動**
- モバイル：人物写真上部→縦書きコピー→吹き出し→商品画像が縦に積まれる
- PC：左に人物写真、右に縦書きコピー + 商品画像の2カラム

**向いている業種**：健康食品、サプリ、化粧品、ベビー用品、食品通販
**参考**：mamarich 葉酸サプリLP

## パターン I：マルチシーン型

```
┌──────────────────────────────┐
│  🐻 ロゴ  サービス名            │
│                                │
│  ┌─────┐   メインコピー  ┌───┐│
│  │シーン │  （複数サイズ・色） │人物││
│  │写真A  │  「自分らしく」    │写真││
│  │（左側）│  「過ごせる時間を」│ B ││
│  │      │    応援！          │   ││
│  └─────┘                 └───┘│
│                                │
│   [バッジ1] [バッジ2] [バッジ3]  │
│                                │
│  手書き風のフォローコピー        │
└──────────────────────────────┘
```

**特徴**
- 2つ以上の人物/シーン写真を左右に配置（利用シーンの多様性を表現）
- メインコピーが中央で、行ごとに色・サイズを変える
- 写真の上にアイコンやイラスト要素が散りばめられる
- 円形バッジ（ギザギザ縁）で信頼ポイントを表示
- 手書き風フォントのフォローコピーで親しみを出す

**CSSの肝**
- 写真はそれぞれ `clip-path` や `border-radius` で形を変える
- テキストの行ごとに `font-size` と `color` を個別指定
- バッジは `border-radius: 50%` + `border` のドット/波線
- 散りばめイラストは小さいSVG/絵文字を `position: absolute` で配置

**レスポンシブ挙動**
- モバイル：左写真→コピー→右写真→バッジの縦積み、または左写真を背景化
- PC：左写真・中央テキスト・右写真の3カラム

**向いている業種**：家事代行、保育、介護、人材、ライフスタイルサービス
**参考**：Bears 暮らしサポートサービスLP

## パターン J：イラスト/アート型

```
┌──────────────────────────────┐
│  ロゴ                   menu  │
│                                │
│         ┌──────────────┐      │
│         │              │      │
│         │  全面イラスト   │      │
│         │  または         │      │
│         │  アートワーク   │      │
│         │              │      │
│         │   手書き風      │      │
│         │   メインコピー   │      │
│         │              │      │
│         │   サブコピー     │      │
│         │  （小さめ複数行）│      │
│         │              │      │
│         └──────────────┘      │
│                          🔊   │
└──────────────────────────────┘
```

**特徴**
- 写真ではなくイラスト/アートワークが画面全体を覆う
- メインコピーは**手書き風フォント**（Zen Kurenaido, Yuji Syuku等）
- テキスト量は控えめ、世界観で引き込む
- ナビはミニマル（ロゴ + ハンバーガーメニュー）
- BGMやサウンド要素と組み合わせることも多い

**CSSの肝**
- イラストは `object-fit: cover` で全面表示
- 手書き風フォント `font-family: 'Zen Kurenaido', 'Yuji Syuku', serif`
- テキストは `mix-blend-mode` でイラストと馴染ませる場合も
- メニューは `backdrop-filter: blur()` でフロスト効果

**レスポンシブ挙動**
- モバイル・PC共通で全面イラスト。テキスト位置だけ微調整
- `object-position` でイラストの見せたい部分を制御

**向いている業種**：教育、NPO、クリエイティブ、ゲーム、カルチャー系サービス
**参考**：Balright（就労支援サービス）

| こういう案件なら | このパターン |
|---------------|------------|
| 専門家の信頼感で売りたい | A：権威キービジュアル型 |
| ユーザーの悩みに寄り添いたい | B：共感ストーリー型 |
| タレント/インフルエンサー起用 | C：セレブリティ型 |
| ビジュアルで世界観を伝えたい | D：フルビジュアル型 |

### 組み合わせ例
- **A+F**：権威型+数字強調（士業で実績数字を大きく）
- **B+H**：共感+商品（悩み→サプリで解決の流れ）
- **G+D**：エディトリアル+フルビジュアル（採用サイトで全面写真+巨大英字）
- **I+E**：マルチシーン+ビフォーアフター（家事代行で悩み→複数解決シーン）
